Features

Everything you need

⇄

Instant Switching

Rotate between unlimited Claude Code accounts. Switch by number or email in a single command.

☰

Usage Stats

Real-time 5-hour and 7-day utilization with reset countdowns. Know exactly when your limits refresh.

🔒

Secure Storage

macOS Keychain integration. File-based storage with 0600 permissions on Linux and WSL.

🏢

Org Support

Manage personal and organization accounts side by side. Same email, different orgs? No problem.

⎌

Safe Rollback

All switch operations use file locking and transaction rollback. If anything fails, nothing changes.

⚡

Zero Dependencies

Single static binary. No runtime, no frameworks, no package managers. Just download and run.

Get Started

Up and running in 60 seconds

1

Download the binary

Grab the latest release for your platform.

# macOS Apple Silicon
curl -Lo cswap https://github.com/jakubciszak/claude-account-swap/releases/latest/download/cswap-darwin-arm64
chmod +x cswap && mv cswap ~/.local/bin/
2

Add your accounts

Log into Claude Code with each account and register it.

# Log in with account 1, then:
cswap --add-account

# Log out, log in with account 2, then:
cswap --add-account
3

Switch away

Rotate to the next account or jump to a specific one.

cswap --switch
cswap --switch-to user@example.com
cswap --list
